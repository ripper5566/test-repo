usage: download_mail_readers.py [-h] [--all]
                                [--platforms PLATFORMS [PLATFORMS ...]]
                                [--list] [--output OUTPUT] [--save-metadata]

Download offline mail reader applications for backup

options:
  -h, --help            show this help message and exit
  --all                 Download all available mail readers
  --platforms PLATFORMS [PLATFORMS ...]
                        Specific platforms to download
  --list                List all available mail readers
  --output OUTPUT       Output directory for downloaded files
  --save-metadata       Save metadata about downloads

Examples:
  # Download all mail readers
  python download_mail_readers.py --all
  
  # Download specific platforms
  python download_mail_readers.py --platforms thunderbird_windows
  
  # List available mail readers
  python download_mail_readers.py --list
  
  # Specify output directory
  python download_mail_readers.py --all --output /path/to/backup
