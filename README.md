# RTL8814AU Driver Reference

## 8814au-20230620

This repository contains references to the RTL8814AU wireless driver.

### Driver Information

- **Chipset**: Realtek RTL8814AU
- **Type**: USB WiFi Adapter Driver
- **Date Reference**: 2023-06-20
- **Protocol**: 802.11ac

### Supported Devices

The RTL8814AU chipset is commonly found in high-performance USB WiFi adapters supporting:
- Dual-band WiFi (2.4GHz and 5GHz)
- 802.11ac Wave 1 and Wave 2
- Up to 1900 Mbps data rates
- Multiple spatial streams (4x4 MU-MIMO)
